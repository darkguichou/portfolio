connexion ( )

connexion à la base de données

La fonction se connecte à la base de données

Parameters
hosthost désigne l'adresse du serveur d'hébergement
dbNamedbName désigne le nom de la base de données
useruser désigne le nom utilisateur
pwdpwd désigne le mot de passe de connexion
Returns
connexion renvoi une connexion à la base de données. Cet outil permet de se connecter à la BDD à chaque fois qu'une fonction en a besoin sans avoir à réécrire les infos nécessaire à la connexion.
